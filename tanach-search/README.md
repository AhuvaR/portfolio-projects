# 📖 TANACH Search Engine API

A .NET Core API that allows users to search through TANACH (Bible) texts. Uses structured text files and follows a clean Three-Layer Architecture.

## ✨ Features
- Structured text parsing into C# models (e.g., `Location`)
- RESTful API endpoints
- Full Swagger API documentation

## 💪 Tech Stack
- C# / .NET Core
- Swagger for API docs
